# InvestmentUIUX by Adminuiux - Angular Mobile App and Admin Dashboard Template

This project is an Angular 19 UI Starter Kit Angular Mobile App and Admin Dashboard template designed to provide a robust and feature-rich foundation for building web applications. It leverages modern UI/UX practices and a comprehensive set of libraries to ensure a high-quality development experience.

## Features

* **Angular 19:** Built with the latest Angular framework for optimal performance and maintainability.
* **Bootstrap 5:** Utilizes Bootstrap 5 for responsive and consistent styling.
* **Responsive Design:** Ensures seamless adaptation across various devices, from mobile phones to tablets.
* **Chart.js:** Implements interactive and customizable charts for data visualization.
* **FullCalendar:** Provides a powerful calendar component for event management.
* **Choices.js:** Enhances select elements with advanced search and selection capabilities.
* **DataTables:** Offers advanced table functionalities, including responsive design and data manipulation.
* **Date Range Picker:** Includes a Bootstrap-integrated date range picker for easy date selection.
* **Dark Mode Support:** Provides a dark mode theme with customizable color options.
* **Swiper.js:** Implements smooth and responsive carousels and sliders.
* **Circle Progress:** Display circular progress bars.
* **Iconic Footer:** Sticky footer menu and informative footer.

## Getting Started

1.  **Open the angular directory:**

    ```bash
    git clone [repository URL]
    cd angular
    ```

2.  **Install Dependencies:**

    ```bash
    npm install
    ```

3.  **Run the Development Server:**

    ```bash
    ng serve
    ```

    Open your browser and navigate to `http://localhost:4200/`.

4.  **Build the Project:**

    ```bash
    ng build
    ```

    This will create a `dist` folder containing the production-ready build.
